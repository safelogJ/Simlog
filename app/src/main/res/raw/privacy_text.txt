Collection and Use of Your Personal Data:<br>
Types of Data Collected: Usage Data<br><br>

Usage Data is collected automatically when you use the Service.<br>
Usage data may include information such as the time and date of application use, time spent in the application,
the number of active SIM cards in the phone,
the name of the SIM card, the signal strength of the SIM card,
the type of data transfer used by the SIM card.<br>
The collected data is used only in the application itself,
including in the user interface.<br>
The slot number and the name of the SIM card are used to display to the user the active SIM cards that the application can work with.<br>
The slot number and the name of the SIM card are used in the name of the files created by the application,
which store the history of changes in the network signal strength and the type of data transfer.<br>
The data in the files is stored in the format (minute number, data transfer type, network signal strength).<br><br>

Storage of your personal data:<br>
The Company does not store your personal or other data.<br>
Usage data is stored on your device in the application folder (/Android/data/com.safelogj.simlog/files/).<br>
Data about saved routers (IP, login, password) is stored in the internal system folder of the application in encrypted form.<br><br>

Deleting collected data:<br>
You have the right to delete all files with collected data in the application folder (/Android/data/com.safelogj.simlog/files/).<br>
You have the right to delete data about saved routers through the application interface. To do this, select the router to be deleted, enter the command remove in the login field, and press the (+) button.<br><br>
